 35: ## 核心特性
 36: 
 37: ### 🔬 科学回测
 38: - **未来函数零容忍** — PIT (Point-In-Time) 数据对齐，财务数据使用 `ann_date` 而非 `end_date`
 39: - **幸存者偏差消除** — 全量历史股票池（含退市），严格按上市/退市日期过滤
 40: - **分层回测** — 按因子分组构建多空组合，评估因子区分度
 41: 
 42: ### 🏦 A股交易规则完整模拟
 43: - **T+1 交易** — 当日买入不可卖出，次日开盘价成交
 44: - **涨跌停顺延** — 涨停无法买入、跌停无法卖出，自动顺延至下一可交易日
 45: - **100 股取整** — 严格遵守 A 股最小交易单位
 46: - **ADV 流动性限制** — 单笔交易不超过日均成交量的可配置比例
 47: - **印花税时间感知** — 自动识别 2023-08-28 前后的印花税率调整
 48: - **ST/\*ST 过滤** — 严格过滤特别处理股票
 49: 
 50: ### 📊 生产级工程
 51: - **多线程数据下载** — AkShare + Baostock 双源自动切换，指数退避重试
 52: - **DuckDB 列式存储** — 千万级行数据查询 <1s
 53: - **数据质量监控** — 8 类数据校验（OHLC 违规/价格异常/成交量归零/复权因子跳变/成分股滞后/停牌日交易/退市后异常数据/上市前历史），问题数据自动写入 `data_quality_alert` 表
 54: - **9 层数据分层架构** — 原始层、复权层、主表层、因子层，严格分离数据口径，历史回溯零污染
 55: - **检查点恢复** — 回测状态序列化，崩溃后可从上次进度继续
 56: - **交易日历** — 内置 2018-2027 年 A 股节假日数据

 63: stock_data_project/
 64: ├── main_v2_production.py          # 主程序入口
 65: ├── run_tests.py                   # 测试运行脚本
 66: ├── requirements.txt               # Python 依赖
 67: │
 68: ├── scripts/                       # 核心模块
 69: │   ├── __init__.py                # 模块导出 (v3.0)
 70: │   ├── data_engine.py             # 数据引擎 — 多线程下载、DuckDB 存储、数据验证

 73: │   ├── execution_engine_v3.py     # 执行引擎 — T+1 交易模拟、涨跌停、ADV 限制
 74: │   ├── backtest_engine_v3.py      # 回测引擎 — 分层回测、检查点恢复
