# V2.0 全栈重构计划 - 工业级/准实盘水平

## 重构目标
从v1.0生产级跃升至工业级/准实盘水平，解决所有已知问题，达到可对接实盘的标准。

## 四大重构维度

### 1. 交易执行层 (Execution Layer)
**核心问题：修复未来函数，实现真实交易模拟**

- [ ] **T+1开盘价成交**
  - 当前：T日收盘信号，T日收盘价成交（未来函数）
  - 目标：T日收盘信号，T+1日开盘价成交
  - 实现：信号生成与执行分离，引入 `execution_date`

- [ ] **涨跌停顺延**
  - 当前：忽略涨跌停限制
  - 目标：涨停无法买入，跌停无法卖出，顺延至下一可交易日
  - 实现：每日检查涨跌停状态，维护待执行订单队列

- [ ] **100股取整**
  - 当前：允许小数股数
  - 目标：A股最小交易单位100股
  - 实现：仓位计算后向下取整到100的倍数

- [ ] **ADV限制 (Average Daily Volume)**
  - 当前：无成交量限制
  - 目标：单日交易量不超过该股票20日平均成交量的10%
  - 实现：引入流动性约束，大订单分拆执行

- [ ] **停牌处理**
  - 当前：简单过滤停牌股
  - 目标：停牌期间保持持仓，复牌后恢复交易
  - 实现：维护持仓状态，复牌后重新评估

### 2. 数据治理层 (Data Governance)
**核心问题：动态股票池、退市处理、PIT时点对齐**

- [ ] **动态股票池**
  - 当前：固定股票列表
  - 目标：每月根据市值、流动性动态筛选可交易股票
  - 实现：`DynamicUniverse` 类，支持规则配置

- [ ] **退市股票处理**
  - 当前：获取退市股票数据但处理不完善
  - 目标：退市前强制平仓，退市后从股票池移除
  - 实现：监控 `delist_date`，提前N日预警并平仓

- [ ] **ST/*ST处理**
  - 当前：简单过滤
  - 目标：ST股票限制买入（可卖出），*ST股票禁止交易
  - 实现：每日检查ST状态，维护可交易清单

- [ ] **财务数据PIT (Point-In-Time)**
  - 当前：使用 `end_date`（报告期），存在未来函数
  - 目标：使用 `ann_date`（公告日），确保数据当时可用
  - 实现：所有财务因子使用PIT对齐，验证无未来函数

- [ ] **数据质量监控**
  - 当前：基础质量检查
  - 目标：实时异常检测，自动修复/报警
  - 实现：数据管道中插入校验节点

### 3. 计算引擎层 (Compute Engine)
**核心问题：向量化重构、性能优化、长表架构**

- [ ] **向量化因子计算**
  - 当前：部分循环计算
  - 目标：所有因子SQL向量化，性能提升50倍
  - 实现：DuckDB Window函数全面替换Pandas循环

- [ ] **DuckDB Window下推**
  - 当前：部分计算在Python端
  - 目标：所有聚合计算下推至DuckDB
  - 实现：优化SQL，减少数据传输

- [ ] **长表架构 (Long Table)**
  - 当前：宽表存储（每只股票一行）
  - 目标：长表存储（每个因子一列，多行）
  - 实现：重构数据模型，支持灵活扩展

- [ ] **增量计算**
  - 当前：全量重新计算
  - 目标：仅计算新增数据
  - 实现：时间戳标记，差异更新

- [ ] **内存优化**
  - 当前：全量加载到内存
  - 目标：分块处理，流式计算
  - 实现：DuckDB流式查询，Generator模式

### 4. 绩效评价层 (Performance Analytics)
**核心问题：成本模型增强、新指标、归因分析**

- [ ] **成本模型精细化**
  - 当前：固定费率
  - 目标：佣金（双边0.03%，最低5元）+ 印花税（卖出0.1%）+ 滑点（双边0.1%）
  - 实现：`TradingCostModel` 类，支持自定义费率

- [ ] **换手率计算**
  - 当前：无
  - 目标：日换手率、月换手率、年均换手率
  - 实现：跟踪每笔调仓，计算双边换手率

- [ ] **信息比率 (Information Ratio)**
  - 当前：无
  - 目标：超额收益 / 跟踪误差
  - 实现：对比基准（沪深300），计算主动风险调整后收益

- [ ] **Tracking Error**
  - 当前：无
  - 目标：组合收益与基准收益的标准差
  - 实现：日度TE年化

- [ ] **Beta**
  - 当前：无
  - 目标：组合对市场指数的敏感度
  - 实现：滚动回归计算Beta

- [ ] **最长回撤恢复天数**
  - 当前：最大回撤值
  - 目标：从回撤低点恢复到新高的最长天数
  - 实现：扩展回撤分析

- [ ] **归因分析 (Attribution)**
  - 当前：无
  - 目标：Brinson归因，分解行业配置与个股选择
  - 实现：行业分类 + 收益分解
